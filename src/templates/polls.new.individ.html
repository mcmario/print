Інструкція


  • Для додавання нового кандидата, виберіть сферу його діяльності:
    • -Для кандидатів, пов'язаних зі сферою ІТ, натисніть кнопку "Кандидат для ІТ"
    • -Для всіх інших кандидатів, натисніть кнопку "Всі інші"
  • Поля позначені * є обов'язковими для заповнення
  • Для додавання дати використовуйте формат ДД ММ РРРР де ДД-число, ММ - номер місяця, РРРР-рік
  • ІПН повинен містити 10 цифр без пробілів
  • Серію паспорта записуйте без пробілів та великими буквами (АА)
  • Номер паспорта або ІД карти записуйте без пробілів (123456789)
  • Телефонний номер записуйте у повному форматі (+380991234567)
  • Для додавання ще однієї освіти натисніть кнопку "додати освіту" та заповніть необхідні поля
  • дані заповнюються з військового квитка або його замінника
  • Вкажіть рівень володіння мовами, зазначеними в таблиці
  • Для додавання ще однієї мови, введіть назву мови в поле "мова", виберіть рівень володіння нею та натисніть кнопку "додати ще мову"
  • Для додавання кожної наступної мови, щоразу виконуйте вищезазначену процедуру
  • Для додавання ще одного досвіду роботи, натисніть кнопку "додати роботу" та заповніть необхідні поля
  • Для додавання ще контактної особи, натисніть кнопку "додати рекомендації" та заповніть необхідні поля
  • Для додавання ще одного члена сім'ї, натисніть кнопку "додати члена родини" та заповніть необхідні поля
  • Для додавання кожного наступного члена сім'ї, щоразу виконуйте вищезазначену процедуру
  • Для дитини поля "місце роботи/навчання" та "посада" є необов'язковими
  • Одна і та ж відповідь для двох запитань не допускається
  • У разі назаповнення поля чи полів які є обов'язковими для заповнення, Вас буде перенаправлено на перший розділ з незаповненим обовєязковим полем
